This is for my application to the Lua Scripter Role.
This TrashBagSpawner.luau file is a ModuleScript and is part of a bigger spawn system.
Trash bins and dumpsters spawn, and when destroyed, drop trash bags. This script handles trash bags.
Rarity and mutation drop tables are integrated.

You can find a demo of this script here, since the game is unpublished.
